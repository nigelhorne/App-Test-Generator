   33: =head1 NAME
   34: 
   35: App::Test::Generator::Mutator - Generate and apply mutation tests
   36: 
   37: =head1 VERSION
   38: 
   39: Version 0.32
   40: 
   41: =head1 DESCRIPTION
   42: 
   43: B<App::Test::Generator::Mutator> is a mutation engine that programmatically
   44: alters Perl source files to evaluate the effectiveness of a project's test
   45: suite. It analyses modules, generates systematic code mutations (such as
   46: conditional inversions, logical operator changes, and numeric boundary
   47: flips), and applies them within an isolated workspace so tests can be
   48: executed safely against each modified variant.
   49: 
   50: By tracking which mutants are killed (cause tests to fail) versus those that
   51: survive (tests still pass), the module enables calculation of a mutation
   52: score, providing a quantitative measure of how well the test suite detects
   53: unintended behavioural changes.
   54: 
   55: =head2 new
   56: 
   57: Construct a new Mutator for a given source file.
   58: 
   59:     my $mutator = App::Test::Generator::Mutator->new(
   60:         file           => 'lib/My/Module.pm',
   61:         lib_dir        => 'lib',
   62:         mutation_level => 'full',
   63:     );

  185: =head2 prepare_workspace
  186: 
  187: Prepare an isolated temporary workspace for a single mutation test run.
  188: 
  189: The entire C<lib_dir> tree is copied into the workspace so that all module
  190: dependencies resolve correctly when the test suite runs against the mutant.
  191: Only after this copy is complete is the single target file overwritten by
  192: C<apply_mutant>.
  193: 
  194:     my $workspace = $mutator->prepare_workspace();
  195:     $mutator->apply_mutant($mutant);
  196:     local $ENV{PERL5LIB} = "$workspace/lib";
  197:     my $survived = (system('prove', 't') == 0);
  198: 
  199: =head3 Arguments
  200: 
  201: None beyond C<$self>.
  202: 
  203: =head3 Returns
  204: 
  205: A string containing the absolute path to the temporary directory created.
  206: The directory is automatically removed when the object goes out of scope
  207: via L<File::Temp>'s C<CLEANUP =E<gt> 1> behaviour.
  208: 
  209: =head3 Side effects
  210: 
  211: Creates a temporary directory. Recursively copies C<lib_dir> into it.
  212: Sets C<< $self->{workspace} >> and C<< $self->{relative} >>.
  213: 
  214: =head3 Notes
  215: 
  216: Call C<prepare_workspace> once per file, then C<apply_mutant> once per
  217: mutant within that file. Do not store the returned path beyond the
  218: lifetime of the enclosing scope.

  236: sub prepare_workspace {
  237: 	my $self = $_[0];
  238: 
  239: 	# Create a self-cleaning temporary directory
  240: 	my $tmp = tempdir(CLEANUP => 1);
  241: 
  242: 	# Derive the file's path relative to lib_dir for use by apply_mutant
  243: 	my $relative = $self->{file};
  244: 	$relative =~ s/^\Q$self->{lib_dir}\E\/?//;
  245: 
  246: 	# Copy the entire lib tree so all dependencies resolve in the workspace
  247: 	dircopy($self->{lib_dir}, File::Spec->catfile($tmp, $self->{lib_dir}))
  248: 		or croak "dircopy failed: $!";
  249: 
  250: 	$self->{workspace} = $tmp;
  251: 	$self->{relative}  = $relative;
  252: 
  253: 	return $tmp;
  254: }
