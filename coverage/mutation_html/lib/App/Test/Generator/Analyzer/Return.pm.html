   65: =head2 analyze
   66: 
   67: Scan the source code of a method for return patterns and add weighted
   68: evidence to the method object. Detects three patterns: returning a
   69: property from C<$self>, returning C<$self> itself, and returning a
   70: constant literal value.
   71: 
   72:     my $analyser = App::Test::Generator::Analyzer::Return->new;
   73:     $analyser->analyze($method);
   74: 
   75:     my $type = $method->resolve_return_type;
