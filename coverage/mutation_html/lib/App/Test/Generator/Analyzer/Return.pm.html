   66: =head2 analyze
   67: 
   68: Scan the source code of a method for return patterns and add weighted
   69: evidence to the method object. Detects three patterns: returning a
   70: property from C<$self>, returning C<$self> itself, and returning a
   71: constant literal value.
   72: 
   73:     my $analyser = App::Test::Generator::Analyzer::Return->new;
   74:     $analyser->analyze($method);
   75: 
   76:     my $type = $method->resolve_return_type;
   77: 
   78: =head3 Arguments
   79: 
   80: =over 4
   81: 
   82: =item * C<$method>
   83: 
   84: Normally an L<App::Test::Generator::Model::Method> object; evidence is
   85: added to it in place via C<add_evidence>. A plain hashref with a
   86: C<source> or C<body> key is also accepted defensively (as SchemaExtractor
   87: callers may pass one), but in that case no evidence is recorded â€” only
   88: a Model::Method object exposing both C<source()> and C<add_evidence()>
