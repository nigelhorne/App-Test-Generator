   65: =head2 mutate
   66: 
   67: Walk a PPI document and generate one mutant for each non-bare return
   68: statement, replacing its expression with C<undef>. For example,
   69: C<return $result> becomes C<return undef>.
   70:

   72: undef - mutating them would produce a redundant mutant that can never
   73: be killed.
